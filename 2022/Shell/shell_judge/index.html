测试语法:

若为假, 判断返回 1

数值比较:

操作符:
-gt 大于

-ne 不等于
-ge 大于等于
-le 小于等于

if 语法结构

流程控制:

  1. if 如果
  2. then 那么
  3. else 否则

文件测试

操作符:
-f 判断文件是否存在, 是则返回真
-b 判断文件是否是块文件, 是则返回真
-d 判断是否是目录, 是则返回真

-w 判断指定文件或目录是否可写
-x 判断指定文件或目录是否可执行
-e 判断指定文件或目录是否存在
......
