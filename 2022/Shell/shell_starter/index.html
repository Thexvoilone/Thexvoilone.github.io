什么是 shell

shell: 壳(计算机壳层), 命令解释器

第一个 shell 脚本

shell 脚本的四种执行方式:

  1. bash first_shell.sh
  2. sh first_shell.sh
  3. . first_shell.sh
  4. source first_shell.sh

1 和 2 的执行方式, 是子 shell 执行(启动新的 shell 执行)
3 和 4 的执行方式, 是本 shell 执行

shell 变量

用一个固定的字符串去表示不固定的内容, 便于修改

  1. 自定义变量
    定义变量: 变量名=变量值
    变量不能以数字开头
    等号左右不能有空格

推荐变量名命名大于 5 个字母

  1. 调用变量
    若需要提取变量的内容时, 需要在变量名之前加入 “$” 符号
    可以通过 echo 命令输出变量的值
  1. 变量查看
    在不知道有什么 shell 变量时, 可以进行变量查看
  1. 变量删除
    若不想使用一个变量, 就可以删除该变量
  1. 编写脚本
    然后授权脚本

三目表达式:
&& 和 || 是三目表达式
&& 前面的命令成功, 执行后面的命令
|| 前面的命令失败, 执行后面的命令

  1. 交互定义变量
    read 从键盘读入变量名
    赋值方式: read 变量名

3 种引号:

  • "" 弱引用
  • '' 强引用
  • `` 优先执行
