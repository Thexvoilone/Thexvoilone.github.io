高级 DOM 操作

DOM 概述

DOM 是连接 js 和 浏览器 的接口

  • DOM 允许我们使用 js 与浏览器互动
  • 利用 js 增删改查 HTML 元素和属性, 以及 CSS 样式
  • DOM 树基于 HTML 文件生成
  • DOM 是一个非常复杂的 API

API(Application Programming Interface)

DOM API 的结构

事件: 捕获和冒泡

事件传播分三个阶段

  • 捕获
  • 目标
  • 冒泡

事件监听器 .addEventListener() 默认只监听冒泡事件

若要事件监听器捕获1事件, 需额外添加第三个参数 true :
.addEventListener('事件', 回调函数, true)
