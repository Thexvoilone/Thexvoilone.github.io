概念

  • 模块
    • 简单来讲, 模块是单独的 py 文件
  • 包
    • 一个有层次的文件命令结构, 内部可以包含多个模块和子包
  • 库
    • 完成一定功能代码的集合
    • 可以是模块, 也可以是包

包和模块

创建

模块就是 .py 文件

包的创建, 需要在目录中创建一个名为 __init__.py 文件1
一般都建议创建 __init__.py 文件

__init__.py作用:

  • 导入包的时候, 会自动执行 __init__.py
    可以做一些包的预操作
